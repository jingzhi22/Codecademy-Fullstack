CSS auto keyword

Dealing with overflow

Height and Width Maximums/Minimums

  margin: auto;
}
 
small-block {
  overflow: scroll;
}
 
/* Any element with class "column" will be at most 200 pixels wide, despite the width property value of 500 pixels. */

                
.column {
  max-width: 200px;
  width: 500px;
}
